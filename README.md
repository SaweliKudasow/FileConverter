# 📁 File Converter

Web app to convert **images**, **text**, **audio**, and **video** via drag-and-drop. Backend: Express + FFmpeg; UI is in German.

## ⚙️ Requirements

- **Node.js** (v14+)  
- **FFmpeg** on your `PATH` (for audio/video and image pipeline)  

## 🚀 Quick start

```bash
git clone https://github.com/SaweliKudasow/FileConverter.git
cd FileConverter
npm install
npm start
```

Open **http://localhost:3000** (or your `PORT`).

## 📌 Usage

1. Pick a file → 2. Choose target format → 3. Convert → 4. Download.
